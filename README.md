# Proyecto-Módulo-1-Fundamentos

Aplicación web desarrollada en Streamlit que integra conceptos fundamentales de programación en Python como variables, condicionales, estructuras de datos, funciones, programación funcional y Programación Orientada a Objetos (POO).

Este proyecto forma parte del Trabajo Práctico del Módulo Python Fundamentals y tiene como objetivo demostrar la aplicación práctica de los fundamentos de programación mediante una interfaz interactiva.

---

## Tecnologías utilizadas

- Python 3
- Streamlit
- Git
- GitHub

---

## Estructura del proyecto

El proyecto está compuesto por un archivo principal:

- `app.py` → contiene toda la lógica y la interfaz de la aplicación
- `README.md` → documentación del proyecto

---

## Funcionalidades

La aplicación cuenta con un menú lateral que permite navegar entre las siguientes secciones:

### Home
Presenta la información general del proyecto, el objetivo y las tecnologías utilizadas.

### Ejercicio 1 – Variables y Condicionales
Permite ingresar un presupuesto y un gasto, y evalúa si el gasto se encuentra dentro del presupuesto utilizando estructuras condicionales.

Conceptos aplicados:
- Variables
- Inputs
- Condicionales (`if`, `else`)
- Operadores

---

### Ejercicio 2 – Listas y Diccionarios
Permite registrar actividades mediante formularios y almacenarlas en una lista de diccionarios.

Conceptos aplicados:
- Listas
- Diccionarios
- Estructuras de datos
- Manejo de datos en memoria (`st.session_state`)

---

### Ejercicio 3 – Funciones y Programación Funcional
Calcula el retorno esperado de las actividades registradas utilizando funciones y programación funcional.

Conceptos aplicados:
- Funciones
- Parámetros
- `map()`
- `lambda`

---

### Ejercicio 4 – Programación Orientada a Objetos (POO)
Utiliza una clase `Actividad` para representar objetos con atributos y métodos.

Conceptos aplicados:
- Clases
- Objetos
- Métodos
- Encapsulamiento
